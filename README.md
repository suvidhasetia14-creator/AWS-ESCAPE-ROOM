#  AWS Escape Room – Solutions Architect Associate (SAA-C03) Challenge

##  Overview

This repository documents my completion of the *AWS Escape Room: Exam Prep for AWS Certified Solutions Architect – Associate (SAA-C03)* offered by AWS Training & Certification.

The AWS Escape Room is a hands-on, gamified learning experience designed to strengthen practical cloud skills by solving real-world challenges across various AWS services and architectural scenarios.

Through this challenge, I gained experience in designing, deploying, securing, and troubleshooting cloud solutions while following AWS best practices.


##  Objectives

- Strengthen understanding of core AWS services.
- Apply AWS architectural best practices.
- Gain hands-on experience with cloud infrastructure.
- Improve problem-solving and troubleshooting skills.
- Prepare for the AWS Certified Solutions Architect – Associate (SAA-C03) certification.
